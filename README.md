# Task-2-codsoft
🎯 Smart Tic-Tac-Toe AI Arena

📖 Project Description

Smart Tic-Tac-Toe AI Arena is a Python-based strategic board game where players compete against an intelligent computer opponent. The game features AI-powered moves, move history tracking, an undo system, and a clean console interface designed to enhance the gaming experience.

This project demonstrates the implementation of game logic, decision-making algorithms, and user interaction using Python programming.

🌟 Key Features

✅ Human vs AI Gameplay

✅ Beginner-Friendly and Easy to Understand

---

🎮 How to Play

1. Run the Python file.
2. Enter a position number from 1 to 9.
3. The AI will automatically make its move.
4. Type U to undo the previous round of moves.
5. Continue until a player wins or the game ends in a draw.

Position Reference

1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9
